# README: VoiceLoop Hook

## Огляд
Цей хук (`useVoiceLoop`) реалізує циклічний процес «слухай — відтворюй» для створення ефекту «папуги» або інтерактивного персонажа. Він автоматично детектує голос користувача, записує його, змінює тональність (pitch) і відтворює назад через зовнішній динамік.

## Технологічний стек
- `expo-audio-studio` — низькорівнева робота з аудіо; дозволяє отримувати потік PCM у реальному часі для розрахунку децибел (RMS) без переривання запису.
- `expo-av` — відтворення записаних файлів та керування аудіо-режимом пристрою.
- `rive-react-native` — синхронізація стану аудіо з анімацією персонажа через StateMachine.


## Опис
- `expo-audio-studio` — використовував для кращого контролю над звуком
- `rive-react-native` — використовував більш стару версію, ніж зараз доступна (@rive-app/react-native), оскільки в мене старий Xcode 16.2
- `npx create-expo-app@latest` — проєкт створювався звичайним способом для Expo
